##项目名称：
图书排行榜

##上手指南：
以下指南将帮助你在本地机器上安装和运行该项目，进行开发和测试。

##安装要求，需要以下环境：

- Git
- Node

>首先你需要
### 克隆项目
```git clone https://github.com/MrChen2000/book.git```
>然后
### 进入项目目录
```cd book/book-client```
>再
### 安装依赖
```npm install```
>最后启动项目
### 启动服务
```npm run dev```

以上是客户端的

>服务端：

### 进入项目目录
```cd book/BookServer```

### 启动服务
npm start


###打包部署
